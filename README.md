> 快捷键一按，AI 即来。说"打开微信"就能打开，说"查看IP"就能执行。
> 纯 Python，零 Electron，零 Node.js。

**Features:**
- 🏝️ 灵动岛 — 屏幕顶部浮动状态胶囊
- 🐱 像素橘猫 — 72 帧动画宠物
- ⚡ 自然语言命令 — 说人话就能执行
- 🤖 Claude Code 集成 — 零配置直接用
- 🔌 多后端 — DeepSeek / NVIDIA / Ollama 等

## 🔮 RAG 知识库智能问答系统

基于 RAG 架构的企业级知识库问答系统，支持文档上传、智能分块、语义检索与多 Agent 协作生成。

**技术栈:** Python + FastAPI + ChromaDB + DeepSeek V3
